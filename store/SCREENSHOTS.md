# Screenshots Needed for Chrome Web Store

CWS requires 1-5 screenshots (1280×800 or 640×400).

## Recommended Screenshots

1. **Popup UI — Default view**
   - Open extension popup on YouTube with subtitles visible
   - Show the clean dark-themed settings panel with preset dropdown
   - Subtitles styled with "Recommended" preset visible in background

2. **YouTube — Before/After**
   - Side by side or sequential showing default YouTube subtitles vs styled
   - Highlight drop shadow, custom colors, font change

3. **Per-site settings**
   - Show the scope toggle (All Sites / YouTube) in the popup
   - Demonstrate that different platforms can have different styles

4. **Multiple platforms**
   - Collage or split view showing styled subtitles on YouTube, Nebula, Dropout, Prime Video, Max
   - Shows breadth of platform support

5. **Preset selection**
   - Popup with preset dropdown open showing Recommended, Classic, Minimal options

## Notes

- All screenshots must be taken on Jacek's machine with the extension loaded
- Use incognito or a clean profile to avoid other extensions cluttering the UI
- Capture at 1280×800 for best CWS display
- Ensure subtitle text is clearly readable and the styling difference is obvious
